## UI/UX Internship Task 4

# Close – Sales CRM SaaS UI Design

## 📌 Project Overview
Close is a modern Sales CRM SaaS UI/UX design created in Figma. The platform is designed to help sales teams and businesses manage leads, track customer interactions, monitor pipelines, and improve overall sales performance through a clean and efficient dashboard interface.

## 🎯 Design Goal
The goal of this project was to design a powerful yet simple CRM system that improves sales workflow efficiency, enhances lead management, and provides clear visibility into sales pipelines and performance analytics.

## 🧩 Pages Designed
- Login Page
- Sign Up Page
- Dashboard Overview
- Leads Management Screen
- Deals / Pipeline View
- Customer Details Page
- Tasks & Activity Tracker
- Analytics Dashboard
- Reports Section
- Settings Page
- Interactive Prototype Flow

## ✨ Key UX Decisions
- Clear and structured dashboard layout
- Easy lead tracking and management system
- Visual sales pipeline representation
- Quick access to customer data
- Minimal and professional UI design
- Data-focused layout for better decision making
- Smooth navigation between CRM modules

## 🎨 Color & Typography
- Professional and corporate color scheme
- Clean and readable typography
- Strong visual hierarchy for data-heavy screens
- Consistent spacing for better usability

## 🛠 Tools Used
- Figma
- Auto Layout
- Components & Variants
- Design System
- Prototyping Tools
- Icons & UI Assets

## 🚀 Outcome
The final design provides an efficient CRM experience that helps sales teams manage leads, track deals, and analyze performance in a structured and user-friendly interface.
